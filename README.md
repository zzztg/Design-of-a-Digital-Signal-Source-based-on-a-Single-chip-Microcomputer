# 双路 DDS 数字信号源

基于 **STM32F407VGT6** + **2× AD9834** 的双通道 DDS 信号发生器。

| 参数     | 范围            |
| -------- | --------------- |
| 频率     | 1 Hz ~ 10 MHz   |
| 幅值     | ±18 V           |
| 相位     | 0° ~ 360° (0.1°分辨率) |
| 波形     | 正弦波 / 三角波 / 正弦波+方波 |

## 硬件架构

```
                    ┌───────────────────────────────────┐
                    │           STM32F407VGT6            │
                    │  TIM2 (编码器) ← ENC1 频率         │
                    │  TIM3 (编码器) ← ENC2 幅值         │
                    │  TIM4 (编码器) ← ENC3 相位         │
                    │                                   │
                    │  SPI2 ──→ AD9834_A (CS=PB12)       │
                    │  SPI2 ──→ AD9834_B (CS=PB11)       │
                    │  SPI2 ──→ DAC (幅值控制)            │
                    │  I2C1 ──→ OLED 128×64              │
                    │  USART1 ─→ CH340 (调试串口)         │
                    └───────────────────────────────────┘
```

**幅值通道**: AD9834 输出 (~0.6Vpp) → 可编程增益放大器 → 功率放大器 → ±18V

### 主要元器件

| 器件            | 型号            | 说明              |
| --------------- | --------------- | ----------------- |
| 主控 MCU        | STM32F407VGT6   | Cortex-M4, 168MHz  |
| DDS 芯片        | AD9834 × 2      | 75MHz 主频, 28位频率字 |
| 旋转编码器      | EC11 × 3        | 频率/幅值/相位调节      |
| 显示屏          | OLED 128×64     | I2C 接口, 实时参数显示  |
| 调试接口        | CH340 (USART1)  | 115200, 8N1         |

## 软件设计

### AD9834 驱动 (BSP)

使用位域联合体映射 AD9834 控制寄存器，通过"影子寄存器"维护芯片状态：

```c
typedef union {
    uint16_t value;
    struct {
        uint16_t mode    : 1;   // 模式: 0=正弦, 1=三角
        uint16_t div2    : 1;   // 方波分频
        uint16_t opbiten : 1;   // 方波使能
        uint16_t reset   : 1;   // 复位
        uint16_t b28     : 1;   // 28位写入模式
        // ... 其他控制位
    };
} ad9834_control_register_t;
```

API:

- `AD9834_Init()` — 初始化实例
- `AD9834_SetFrequency()` — 写频率寄存器 (28位频率字)
- `AD9834_SetPhase()` — 写相位寄存器 (12位相位字)
- `AD9834_SetOutputWave()` — 选择波形 (正弦/三角/正弦+方波)
- `AD9834_SetReset()` — 复位/释放复位
- `AD9834_SelectFrequencyRegOutput()` / `AD9834_SelectPhaseRegOutput()` — 寄存器输出选择

### 系统主循环

系统采用状态机驱动的事件循环：

1. **读取编码器增量** — TIM2/3/4 编码器模式捕获旋转
2. **扫描按键** — 10ms 消抖状态机，切换编辑位 (digit)
3. **参数调节** — 按位编辑 (支持 1/10/100/1k/10k/100k/1M 步进)
4. **硬件更新** — 脏标记驱动 AD9834 和 DAC 写入
5. **OLED 刷新** — 100ms 间隔，显示频率/幅值/相位
6. **空闲休眠** — 无事件时 `__WFI()` 等待中断

### 系统时钟

```
HSE (8MHz) → PLL → SYSCLK = 168MHz
  ├── AHB  = 168MHz
  ├── APB1 = 42MHz  (TIM2/3/4)
  └── APB2 = 84MHz  (SPI2, USART1)
```

## 交互方式

| 操作              | 功能                          |
| ----------------- | ----------------------------- |
| **ENC1 旋转**     | 调节频率 (按 digit 位权步进)    |
| **ENC2 旋转**     | 调节幅值                       |
| **ENC3 旋转**     | 调节相位                       |
| **ENC1 按键**     | 切换频率编辑位 (1Hz→10Hz→...→1MHz) |
| **ENC2 按键**     | 切换幅值编辑位 (10mV→100mV→1V→10V) |
| **ENC3 按键**     | 切换相位编辑位 (0.1°→1°→10°→100°) |
| **OLED 显示**     | 实时显示当前通道参数 + 编辑位光标    |
